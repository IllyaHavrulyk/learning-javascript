Можно ли изменить объект, объявленный с помощью const? Как вы думаете?

const user = {
  name: "John"
};

// это будет работать?
user.name = "Pete";

Відповідь: змінну user, яка зберігає ссилку на об'єкт, не можна змінити,
але вміст об'єкта можна зміннювати, тобто стрічка user.name = "Pete"; - спрацює
